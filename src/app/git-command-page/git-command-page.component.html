GitHub Commands

This section is divided into 3 parts
Initializing Git Exsisting Project
Push a update of the Project
Deploying Project using Ghpages on Github.

Initializing Git Exsisting Project

1. Go to GitHub and Signup.
2. Go to your Terminal or Command-Prompt and localte your directory.
3. Initialize empty git repository.
4. Git commands for display files which are created newly or edited files but not commited yet.
5. Git commands to something all files / folders.
6. Git command to commit a file with a message.
7. Next, add remote origin.
8. In this command we change the git branch from master to main since git has made some changed.
9. Finally push project on github.
Initialize
Display project files status
To something
To commit file with message
Adding remote origin
Git branch
Push project on github 
git init
git status
git add --all
git commit -m "message-to-commit"
git remote add origin https://github.com/git_username/git_repositoryname.git
git branch -M main
git push -u origin main

Deploying Project using Ghpages on Github

1. Go to your Terminal or Command-Prompt and localte your directory.
2. Read and Execute all commands from the previous section to make sure git repository is up-to-date.
3. Install Ghpages using npm.
4. Next command is to build the project in production mode.This command generates a Dist directory.
5. Finally this command (explain)
Install Ghpages
Build Project
NGH Command 
npm install -g angular-cli-ghpages
ng build --prod --base-href "hhtps://git_username.github.io/git_repositoryname"
ngh --dir dist/folder_name
